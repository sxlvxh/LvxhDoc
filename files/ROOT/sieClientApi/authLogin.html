授权码登录

客户端登录系统调用接口

接口协议

HTTP+JSON

请求路径 

http://127.0.0.1:39290/platWeb/three/service.action

请求参数说明

methodString登录方法 【固定传 "authLogin"】
paramsObject授权码登录参数
authCodeString授权码

请求参数实例

{
	"method": "authLogin",
	"params": {
		"authCode": "12356"
	}
}

返回参数说明

codeint返回码
descString返回描述
resultObject返回结果
menuObject 数组菜单数组
menuCodeString菜单编号
nameString菜单名称
menuActionString菜单点击跳转的URL
imgUrlString菜单图标URL地址
configObject系统配置
sieIpString流媒体IP地址
siePortint流媒体端口号
domainCodeString流媒体域编码
userObject用户信息
nameString用户名称
imgUrlString用户头像
userCodeString全系统唯一用户标识
