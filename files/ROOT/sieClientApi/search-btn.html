查询按钮参数说明

  略

参数说明

url查询接口地址
sort排序规则
css按钮样式
img按钮图标样式
paging是否加载分页插件
enableCheck多选框显示规则

参数实例

 {
	"url": "../platuserlist/grid.action",
	"sort": {
		"field": "id",
		"order": "desc"
	},
	"css": "btn btn-white btn-success",
	"img": "glyphicon glyphicon-search btn-white",
	"paging": true,
	"enableCheck": [{
			"field": "id",
			"rule": [{
					"rule": "equal",
					"value": 1
				}
			]
		}
	]
}
