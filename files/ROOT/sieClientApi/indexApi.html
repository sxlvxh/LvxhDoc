业务系统接口

1. 业务服务器在影视频客户端配置业务服务器唯一请求地址
2. 接口鉴权，使用Session校验模式
3. 除登录接口之外的接口均需要在消息头增加SessionID参数

接口协议

HTTP+JSON

字符编码

请求及返回结果都使用UTF-8字符集进行编码。

请求路径 

http://127.0.0.1:39290/platWeb/three/service.action

参数说明

method接口路由方法 【业务服务器根据此方法来处理相应的业务逻辑】
params根据接口路由方法配置不同的参数

method参数说明

accountLogin账号登录参数说明
authLogin授权码登录参数说明
memberList根据业务码获取参会成员列表需要sessionID参数说明
memberDetail根据用户码获取参会成员详情需要sessionID参数说明

请求头参数说明

http header 增加 Cookie: JSESSIONID=DFEEF6EE6A0F3D0ED230CC0ECBDEFE0E 此sessionID为登录返回的SessionID

请求参数实例

 {
	"method": "accountLogin",
	"params": {...}
}

返回参数实例

 {
	"code": 0,
	"desc": "成功！",
	"result": {...}
}
