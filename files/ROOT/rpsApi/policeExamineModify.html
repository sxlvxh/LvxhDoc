修改警力报备记录

 修改警力报备记录

接口协议

HTTP+JSON

字符编码

请求及返回结果都使用UTF-8字符集进行编码。

请求路径 

http://192.168.2.107:8080/rps-war/rpspolicereport/update.action

参数说明

id记录id(可不用显示)
policeRecType报备类型 0：签到 1 工作日志 通过查询数据字典获取
areaType岗位类型 通过查询数据字典获取
reportDesc报备日志内容

请求参数实例

{
    "id":"1",
    "policeRecType": "0",
    "areaType": "0",
    "reportDesc": "rr"
}

返回参数实例

{
    "code": 0,
    "desc": "更新成功"
}
