修改监督检查

 修改监督检查

接口协议

HTTP+JSON

字符编码

请求及返回结果都使用UTF-8字符集进行编码。

请求路径 

http://192.168.2.107:8080/rps-war/rpscheck/update.action

参数说明

id记录id(可不用显示)
entCode企业编号(登录人的企业编号，可不用显示)
deptId部门ID(登录人的部门ID,可不用显示)
checkType监测类型：0 默认类型 通过查询数据字典获取
fileList文件列表 数组，多条记录
checkDesc检查内容
checkResp检查结果
checkTime检查时间
checkUserId检查人员id(登录人的用户id，可不用显示)

请求参数实例

{
    "id":"1",
    "entCode": "10000",
    "deptId": "10200020",
    "checkType": "0",
    "fileList": [{"fileUrl":"d/ss/sss"},{"fileUrl":"c/ss/sss"}],
    "checkDesc": "本次检查涉及XXXXXX",
    "checkResp": "本次检查最终结果是A+",
    "checkTime": "2020-05-19 10:10:10",
    "checkUserId": "1"
}

返回参数实例

{
    "code": 0,
    "desc": "更新成功"
}
