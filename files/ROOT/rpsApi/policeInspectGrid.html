分页查询监督检查

 分页查询监督检查

接口协议

HTTP+JSON

字符编码

请求及返回结果都使用UTF-8字符集进行编码。

请求路径 

http://192.168.2.107:8080/rps-war/rpscheck/gridEx.action

参数说明

page分页对象
curPage当前页码
pageSize每页查询记录条数

请求参数实例

{
	"pages":{"curPage":1,"pageSize":10}
}

返回参数实例

{
    "code": 0,
    "desc": "分页查询成功",
    "result": [],
    "pages": {
        "curPage": 1,
        "pageCount": 1,
        "pageSize": 10,
        "totalCount": 3,
        "startNo": 0
    }
}
