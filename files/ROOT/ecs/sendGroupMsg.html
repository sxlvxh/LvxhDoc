发送群组消息

 通讯录发送群组消息

接口协议

HTTP+JSON

字符编码

请求及返回结果都使用UTF-8字符集进行编码。

请求路径 


https://192.168.2.132:8280/platWeb/platsms/sendGroupMsg.action

参数说明

smsType消息类型
contentType发送消息内容类型contentType 1 文字 2 图片 3 视频 4 语音
src发送者用户编码
content消息内容（字符串）
target群组的groupCode

请求参数实例

{
	"smsType" : "chat",
	"contentType" : 1,
	"src" : "lvxh",
	"content" : "{\"contentType\":1,\"content\":\"12\"}",
	"target" : "b7712f98f0cb43329118aa790c379966"
}

返回参数实例

{"code":0}
