更新消息状态

更新消息状态

接口协议

HTTP+JSON

字符编码

请求及返回结果都使用UTF-8字符集进行编码。

请求路径 

 https://192.168.2.132:8280/platWeb/platsms/getContactInfo.action

参数说明

target 接受者userCode
src 发送人的userCode
sort 排序
field 排序属性
order 排序方式 asc升序 desc降序

请求参数实例

{
	"src" : "29c31e9fa1f1448583348636fda3e8fe",
	"target" : "8f05e88abcea4e02815a8579041cca28",
	"sort" : {
		"field" : "create_time",
		"order" : "asc"
	}
}

返回参数讲解

obj 返回数据
msgList 消息列表
src 发送者userCode
target 接受者userCode
content 消息内容
status 消息状态
smsType 发送者userCode
contentType 内容类型 1 文字 2 图片 3 视频 4 语音
sendType 0 自己发送 1 自己收到
sourceMsg 源消息内容
userList 用户信息列表
name 用户名
sex 用户性别
imgUrl 用户头像
userCode 用户编码
domainCode 域编码
friend 是否是好友
friendMark 好友备注
