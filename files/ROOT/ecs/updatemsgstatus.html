更新消息状态

接口协议

HTTP+JSON

字符编码

请求及返回结果都使用UTF-8字符集进行编码。

请求路径 

 https://192.168.2.132:8280/platWeb/platsms/updateStatus.action

参数说明

target 接受者userCode
src 发送人的userCode
smsType 消息类型
sort 排序
field 排序属性
order 排序方式 asc升序 desc降序

请求参数实例

{
	"target" : "8f05e88abcea4e02815a8579041cca28",
	"smsType" : "notice",
	"pages" : {
		"curPage" : 1,
		"pageSize" : 20
	},
	"sort" : {
		"field" : "id",
		"order" : "desc"
	}
}

返回参数说明

result 消息列表
src 发送者的userCode
target 接受者的userCode
content 消息内容
status 聊天状态
smsType 消息类型
pages 分页参数
curPage 当前分页
pageCount 分页总数
pageSize 分页大小
totalCount 数据总数
startNo 开始查询的id
