查询系统消息

接口协议

HTTP+JSON

字符编码

请求及返回结果都使用UTF-8字符集进行编码。

请求路径 

 https://192.168.2.132:8280/platWeb/platsms/grid.action

参数说明

target 接受者userCode
smsType 消息类型 notice：系统提示消息
pages 分页数据
curPage 当前分页
pageSize 每页查询的个数

请求参数实例

{
	"target" : "8f05e88abcea4e02815a8579041cca28",
	"smsType" : "notice",
	"pages" : {
		"curPage" : 1,
		"pageSize" : 20
	},
	"sort" : {
		"field" : "id",
		"order" : "desc"
	}
}

返回参数实例

{"code":0}
