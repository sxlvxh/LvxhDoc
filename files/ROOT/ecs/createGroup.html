创建群组与群组添加人员

 通讯录创建群组与群组添加人员

接口协议

HTTP+JSON

字符编码

请求及返回结果都使用UTF-8字符集进行编码。

请求路径 


https://192.168.2.132:8280/platWeb/platcontact/addCustomGroup.action

参数说明

groupCode群组的groupCode（添加群组成员时填写）
contactList成员列表
userCode用户编码
friendMark用户备注（用户名陈）

请求参数实例

创建群组：
{
	"groupCode" : "",
	"contactList" : [{
		userCode: "7272f856b5c74aabb309f0ddc754f375",
		"friendMark": "王春春"
	},{
		userCode: "c91b454168bc4790bc1c515f47b1dd9c",
		"friendMark": "吕孝怀12"
	}]
}

群组增加人员：
{
	"groupCode" : "b7712f98f0cb43329118aa790c379966",
	"contactList" : [{
		userCode: "7272f856b5c74aabb309f0ddc754f375",
		"friendMark": "王春春"
	},{
		userCode: "c91b454168bc4790bc1c515f47b1dd9c",
		"friendMark": "吕孝怀12"
	}]
}

返回参数实例

{"code":0}
