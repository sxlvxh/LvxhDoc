好友发送消息

 通讯录好友发送消息

接口协议

HTTP+JSON

字符编码

请求及返回结果都使用UTF-8字符集进行编码。

请求路径 


https://192.168.2.132:8280/platWeb/platsms/sendMsg.action

参数说明

smsType消息类型
contentType发送消息内容类型contentType 1 文字 2 图片 3 视频 4 语音
src发送者用户编码
content消息内容（字符串）
targets消息接受者的用户编码（数组）
target消息接受者的用户编码

请求参数实例

{
	"smsType" : "chat",
	"contentType" : 1,
	"src" : "3c5d15220cb14d3d97377bb5d5014c4a",
	"content" : "{\"contentType\":1,\"content\":\"1111\"}",
	"targets" : ["7272f856b5c74aabb309f0ddc754f375"],
	"target" : "7272f856b5c74aabb309f0ddc754f375"
}

返回参数实例

{"code":0}
