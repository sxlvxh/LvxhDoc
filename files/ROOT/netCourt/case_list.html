接口说明

查询案件列表

请求路径 【HTTP+JSON】

 http://192.168.2.132:8380/courtWeb/caselist/grid.action

请求参数说明

caseName:案件名陈 （选填）
sort:查询结果排序（选填）field是按什么属性排序， order是排序类型 desc降序 asc升序
pages: 分页查询参数（必填）curPage当前查询页， pageSize单次查询的大小

请求参数示例

{
	"caseName" : "",
	"sort" : {
		"field" : "id",
		"order" : "desc"
	},
	"pages" : {
		"curPage" : 1,
		"pageSize" : 10
	}
}

返回参数说明

result：结果集合
		"createTime" : 创建时间,
		"createUserId" : 1,
		"nofuzzy" : 0,
		"id" : 21,
		"isdel" : 0,
		"updateTime" : 修改时间,
		"updateUserId" : -1,
		"negativeFields" : [],
		"caseCode" : 案件编号
		"caseName" : 案件名陈
		"caseType" : "1",
		"depCode" : 部门编码
		"untakerCode" : 承办人编号
		"syncNo" : 同步编号
		"depName" : 承办部门
		"untakerName" : 承办人
		"caseTypeName" : 案件类型
 "pages": 分页查询对象
        "curPage": 1,当前分页
        "pageCount": 1,一共多少分页
        "pageSize": 10,分页大小
        "totalCount": 3,查询结果共多少条
        "startNo": 0 从第几条数据开始查询
    }
