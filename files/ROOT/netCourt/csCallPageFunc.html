接口说明

按用户查询庭审列表

请求路径 【HTTP+JSON】

 cs调用具体哪个页面的方法

数据库配置说明

msgType:类型 
menuCodeList:菜单编码集合（menuCode）

配置参数示例

[{
		"msgType" : 1,
		"menuCodeList" : ["5101", "5102"]
	}, {
		"msgType" : 2,
		"menuCodeList" : ["5101", "5103"]
	}, {
		"msgType" : 3,
		"menuCodeList" : ["5106"]
	}
]

配置参数说明

msgType:类型 
		1：开庭
		2： 更改庭审状态
		3：加入会议
		4：接收到举证质证页面的透传消息调用的页面
		5：展示详情页面
		7：指定调用举证质证页面的方法
