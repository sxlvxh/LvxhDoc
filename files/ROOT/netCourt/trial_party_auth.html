接口说明

根据授权码查询庭审信息

请求路径 【HTTP+JSON】

http://192.168.2.132:8380/courtWeb/court/getTrialUserByAuthCode.action

请求参数说明

authCode:用户授权码

请求参数示例

{
	"authCode": 998
}

返回参数说明

roomList：庭室固定设备集合
         "id": 10,     
        "roomNodeName": 庭室名称
        "roomCode": "03067578208f4f6689080e35c2437c67", 庭室编码
        "roomNodeType": 设备类型
        "domainCode": 设备域
        "devCode": 设备编号
        "videoChannelCode": 通道编号
        "videoStreamCode": 流编号
        "videoStreamType": 主副码流标志位 1：主码流 其他为辅码流
        "streamViceFlag": 是否支持手机端副码流观摩 0 是 1 否
partyList： 当事人集合		
        "id": 987,      
        "updateTime": "2020-04-21 16:49:01",    
        "trialCode": 庭审编码
        "seat": 会议画面的顺序
        "userCode": 用户编码
        "verifyStatus": 人脸公安认证标记0 未认证 1 已认证
        "authCode": 鉴权码
        "userName": 当事人名陈
        "loginName": 登录名称
        "mobilePhone": 手机号码
        "entCode": 企业编码（法院编码）
        "seatName": 席位名称
