发送文本消息
请求路径：ppprompt/sendPrompt.action
请求参数描述
属性 类型 备注 说明
caseId int 必须输入 案件ID
caseProcessId int 必须输入 案件活动ID
请求JSON： 
{
	"caseId":11,
	"caseProcessId":8,
	"content":"hello~~~~"
}
返回参数描述：
属性 类型 备注 说明
code int 返回码 0：成功 1：失败
desc String 返回描述
返回JSON： 
{
    "code": 0,
    "result": {
        "id": "1"
    }
}
