案件登记
请求路径：ppcase/registerCase.action
请求参数描述
属性 类型 备注 说明
caseCode String 必须输入 案件代码
caseName String 必须输入 案件名称
caseSource int 必须输入 案件来源
content int 必须输入 案件描述
clerkDepId int 必须输入 书记员所属部门的部门ID
clerkId int 必须输入 书记员的用户ID
clerkName String 必须输入 书记员姓名
undertakerEntCode String 必须输入 承办人所属单位的单位编码
undertakerDepId int 必须输入 承办人所属部门的部门编码
undertakerUserId int 必须输入 承办人的用户ID
undertakerName String 必须输入 承办人的姓名
请求JSON： 
{
  "caseCode": "case_t_0004",
  "caseName": "case name4",
  "caseSource": 5,
  "content": "案件内容",
  "clerkId": 13018625,
  "clerkDepId": 320101,
  "clerkName": "苏Atest001",
  "undertakerEntCode": "320101",
  "undertakerDepId": 320101,
  "undertakerUserId": 1,
  "undertakerName": "yjp"
}
返回参数描述：
属性 类型 备注 说明
code int 返回码 0：成功 1：失败
desc String 返回描述
返回JSON： 
{
    "code": 0,
    "result": {
        "id": "1"
    }
}
