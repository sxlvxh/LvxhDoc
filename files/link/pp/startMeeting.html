开始会议
请求路径：/becsmeeting/notifyMeetingStart.action
请求参数描述
属性 类型 备注 说明
meetingLoginName String 必须输入 发起方登录名称
meetingUserEntCode String 必须输入 发起方企业编码
meetingDomainCode String 必须输入 发起方域编码
meetingName String 必须输入 会议名称
meetingDesc String 会议描述
meetingId int 必须输入 会议编号
请求JSON： 
{
	"meetingLoginName": "root",
	"meetingUserEntCode": "320101",
	"meetingDomainCode": "a84",
	"meetingName": "某某的会议",
	"meetingDesc": "会议描述",
	"meetingTrunkPara": "eyJzdHJVc2VySWQiOiJyb290Iiwic3RyTWVldGluZ0RvbWFpbkNvZGUiOiJhODQifQ==",
	"meetingMode": 2,
	"meetingRecord": 1,
	"meetingInviteStyle": 1,
	"meetingVoiceIntercom": 0,
	"meetingId": 60079196,
	"meetingStartTime": ""
}
返回参数描述：
属性 类型 备注 说明
code int 返回码 0：成功 1：失败
desc String 返回描述
返回JSON： 
{
	"code": 0,
	"desc": "操作成功"
}
