开始接访/案件录像

请求参数描述
属性 类型 备注 说明
caseId int 必须输入 案件ID
activityId int 必须输入 案件当前活动的活动ID
recordSpec
recordType int 必须输入 录像类型
roomId int 必须输入 媒体室ID
roomNodeId int 必须输入 采集点ID
必须输入

{

      "roomNodeId": 11000154
    },
    {
      "recordType": 0,

  ]
}

属性 类型 备注 说明

{
    "code": 0,
    "result": {
        "caseVideoIds": [
