接访登记
请求路径：ppaccept/register.action
请求参数描述
属性 类型 备注 说明
clerkDepId int 必须输入 书记员所属部门的部门ID
localRoomId int 必须输入 本地接访室ID
remoteRoomId int 必须输入 远程接访室ID
acceptedPerson String 必须输入 被接访人姓名
acceptedIDCardNo String 必须输入 被接访人身份证号码
clerkId int 必须输入 书记员的用户ID
clerkName String 必须输入 书记员姓名
undertakerEntCode String 必须输入 承办人所属单位的单位编码
undertakerDeptId int 必须输入 承办人所属部门的部门编码
undertakerUserId int 必须输入 承办人的用户ID
undertakerName String 必须输入 承办人的姓名
请求JSON： 
{
  "acceptedPerson": "LKK1",
  "acceptedIDCardNo":"334585888",
  "clerkId": 13018625,
  "clerkDepId": 320101,
  "clerkName":"苏Atest001",
  "localRoomId": 10100074
}
返回参数描述：
属性 类型 备注 说明
code int 返回码 0：成功 1：失败
desc String 返回描述
返回JSON： 
{
    "code": 0,
    "result": {
        "caseId": 1,
        "activityId":1
    }
}
