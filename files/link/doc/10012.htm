安装GCC补丁及编码器推流说明

一.gcc的安装

将gcc.run包上传至任意目录下：

（1）chmod 755 gcc.run 赋予权限

（2）./gcc.run 执行安装

安装完成后输入命令gcc，出现下图情况即表示安装成功：

没有安装gcc之前，输入gcc命令显示如下：

二.srs的安装

将srs.run包传至任意目录：

（1）chmod 755 srs.run 赋予权限

（2）./srs.run 执行安装

（3）安装完成后，切换到/etc/init.d目录下，如果生成了

srs -> /opt/srs/etc/init.d/srs文件即表示安装成功。

（4）启动srs

切换到/etc/init.d目录下，运行命令/etc/init.d/srs start

若出现 Starting SRS...                                            [  OK  ]

                 SRS started(pid 120717)                        [  OK  ]

即表明安装成功。

三.编码器

1，硬件：

       顶部摄像HDMI线、显示大屏主机HDMI接口、分配器、HDMI线2根，音频线1根、高清编码器(192.168.0.31)、路由器(192.168.1.1)

 

2，接线方式：

顶部摄像HDMI线接入分配器的input口，用HDMI线接入显示大屏主机HDMI接口和分配器的output1口，再用HDMI线接入分配器的output2接口和编码器的input接口上。音频线接入编码器的Audio接口。编码器使用网线直连电脑。

 

3，编码器web端配置：

（1）Pc端ip设为0网段，方可连接编码器。

（2）地址栏输入：192.168.0.31

（3）编码器登录：

用户名：admin 密码：admin
