获取App菜单
请求路径：/login/getAppMenu.action
请求参数描述
属性 类型 备注 说明
id int 必须输入 用户ID
请求JSON： 
{
	"id": 13010285
}
返回参数描述：
属性 类型 备注 说明
code int 返回码 0：成功 1：失败
result List 菜单集合
返回JSON： 
{
	"code": 0,
	"result": [{
			"nofuzzy": 0,
			"id": 41,
			"isdel": 0,
			"name": "APP会议",
			"description": "APP会议",
			"menuUrl": "41",
			"priority": 1,
			"parentId": 0,
			"level": 4,
			"modelId": 1,
			"menuType": 1,
			"userId": 13010285
		}
	]
}
