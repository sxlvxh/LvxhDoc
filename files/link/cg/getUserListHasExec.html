查询电子围栏下有录像的人员
请求路径：/cgtasktemplatetree/getTaskTempUserList.action
接口使用说明： 所有的任务归档查看详情时树形结构使用
请求参数描述
请求JSON： 
{
    "entCode": "320101",   //单位编码，必填
    "deptId": 320101,      //部门ID，非必填
    "cgTemplateType":"1",  //任务类型，1日常任务，2保障任务，3临时任务4上报事件5渣土管控，必填
    "classify":"",          //保障任务类型，cgTemplateType为2时必填，1扫雪防冻2国家公祭日3灯会保障4防汛保障
    "taskDate":"2019-01-16",
    "taskId":17,            //任务ID
    "areaId":174           //区域ID
}
返回参数描述：
属性 类型 备注 说明
code int 返回码 0：成功 1：失败
result List 人员集合
返回JSON： 
{
    "code": 0,
    "desc": "查询成功",
    "result": [
        {
            "id": 1,            
            "entCode": "320101",
            "deptId": 320101,
            "tempId": 1,      //任务模板ID
            "userId": 1,      //用户ID
            "userName": "dsfdsf",
            "empEntCode": "320101",
            "efenceId": "11",          //区域对应电子围栏ID
            "areaTimeId": 17,     //任务区域时间段ID
            "extendCol1": "",
            "extendCol2": "",
            "extendCol3": ""
        }
    ]
}
