接口约定

  •  接口约定

接口协议为 HTTP 协议。

所有接口输入输出参数均使用 String/Json 字符串形式。

返回结果参数类型，类型都为 Json 字符串格式。

  • 消息正文格式说明

所有接口的请求和响应数据按 JSON 格式来封装，如下所示：

{

    "Name": "huaiye",

    "Num": 0,

    "InCity": ["nanjing", "yangzhou"],

    "get_group_channel_list": {

        "GradeSchool": "xinghua",

        "MiddleSchool": ["mahua", "zihua"]

    }

}

  • 字符编码

请求及返回结果都使用 UTF-8 字符集进行编码。

  • 返回码说明

{

    "code": 0,

    "desc": "查询成功",

    "result": [{

                "nofuzzy": 0,

                "id": 10102356,

                "isdel": 0,

                "userId": 1,

                "groupName": "常用联系人",

                "groupType": 2

            }

        ]

    }
