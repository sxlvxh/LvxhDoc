根据任务ID查询电子围栏列表
请求路径：/cgtasktemplatetree/getTaskTempAreaList.action
接口使用说明： 任务管理模块，查看任务详情时，初始化左边的树形第一级调用
请求参数描述
请求JSON： 
{
    "entCode": "320101",   //单位编码，必填
    "deptId": 320101,      //部门ID，非必填
    "cgTemplateType":"1",  //任务类型，1日常任务，2保障任务，3临时任务4上报事件5渣土管控，必填
    "classify":"",          //保障任务类型，cgTemplateType为2时必填，1扫雪防冻2国家公祭日3灯会保障4防汛保障
    "taskDate":"2019-01-14",  //日期
    "taskId":14            //任务ID，所点击记录得ID
}
返回参数描述：
属性 类型 备注 说明
code int 返回码 0：成功 1：失败
result List 电子围栏集合
