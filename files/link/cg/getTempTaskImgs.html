获取临时任务图片
请求路径：/cgtemptaskattach/list.action
请求参数描述
属性 类型 备注 说明
entCode String 必须输入 单位代码
taskId int 必须输入 任务ID
请求JSON： 
{
    "entCode": "320101",
    "taskId": 42
}
返回参数描述：
属性 类型 备注 说明
code int 返回码 0：成功 1：失败
result List 图片路径集合
