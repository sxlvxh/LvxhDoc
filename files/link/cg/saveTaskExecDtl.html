保存任务执行明细
请求路径：/cgtasktempexeclist/saveTaskExecDtl.action
请求参数描述
属性 类型 备注 说明
entCode String 必须输入 单位代码
taskTempId int 必须输入 任务模板ID
tempAreaId String 必须输入 任务区域Id
tempAreaTimeId String 必须输入 任务区域时间段Id
userId String 必须输入 用户Id
userName String 必须输入 用户名称
loginName String 必须输入 用户登录名
execName String 必须输入 执行名称
execNote String 必须输入 执行备注
detailList String 必须输入 执行明细
请求JSON： 
{
    "taskTempId": 1,
    "entCode": "320101",
    "tempAreaId": 22,
    "tempAreaTimeId": 31,
    "userId": 13010283,
    "userName": "陆浩",
    "loginName": "C0F2D6DC942E4EC9B4DB70059C97DC47",
    "execName":"",
    "execNote":"",
    "detailList":[{
    	"contentType":"1",   //明细内容类型
    	"id":1,              //明细内容id，删除时必须
    	"extendCol1":1,       //是否删除，1删除，0不删除
    	"src":"C0F2D6DC942E4EC9B4DB70059C97DC47",   //用户登录名称
    	"content":"1545276670296.mp3",   //文件名称
    	"extendCol2":"3"     //录音文件时长
    }]
}
