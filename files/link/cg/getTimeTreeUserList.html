时间树查询执行人列表
请求路径：/cgtasktemptree/getTimeTreeUserList.action
请求参数描述
属性 类型 备注 说明
entCode String 必须输入 单位代码
deptId int 非必填 部门ID
taskTempId int 必填 任务模板ID
areaTimeId int 必填 任务时间段ID
请求JSON： 
{
    "entCode": "320101",
    "deptId": 320101,
    "areaId":18,
    "startTime": "08:00",
    "endTime": "09:00"
}
返回参数描述：
属性 类型 备注 说明
code int 返回码 0：成功 1：失败
result List 执行人集合
