根据围栏类型查询人员执行明细
请求路径：/cgtasktempexeclist/getUserExecDtlByAreaType.action
请求参数描述
属性 类型 备注 说明
entCode String 必须输入 单位代码
areaType String 必填 围栏类型
efenceId int 必填 电子围栏ID
taskDate String 非必填 任务日期
userId int 必填 用户Id
loginName String 必填 用户登录名
请求JSON： 
{
	"entCode":"320101",
	"areaType":"2",
	"efenceId":17,
	"taskDate":"2018-12-25",
	"userId":13014598,
	"loginName":"8816490587FB4E999E66209EC9FCCE7E"
}

code int 返回码 0：成功 1：失败
result List 人员执行明细集合
