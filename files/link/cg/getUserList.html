获取部门人员（此接口可以查询企业下的某个部门的人员，也可以查询所有人员）
请求路径：/sysentuserlist/getUserList.action
请求参数描述
属性 类型 备注 说明
entCode String 企业编号
depId String 部门编号
请求JSON： 
{
	"entCode": "320114",
	"depId": "320114"
}

属性 类型 备注 说明
code int 返回码 0：成功 1：失败

desc String 返回描述
nState int 用户状态 0不在线 1：空闲，2采集中，3：对讲中 4：会议中 在线状态
