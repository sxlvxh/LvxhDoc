非日常任务获取电子围栏列表
请求路径：/cgtasktemptree/getPostTreeAreaList.action
接口使用说明： 非日常任务地图右侧岗点树使用
请求参数描述
属性 类型 备注 说明
entCode String 必须输入 单位代码
deptId int 非必填 部门ID
请求JSON： 
{
    "entCode": "320101",   //单位编码，必填
    "deptId": 320101,      //部门ID，非必填
    "cgTemplateType":"1",  //任务类型，1日常任务，2保障任务，3临时任务4上报事件5渣土管控，必填
    "classify":""          //保障任务类型，cgTemplateType为2时必填，1扫雪防冻2国家公祭日3灯会保障4防汛保障
}
返回参数描述：
属性 类型 备注 说明
code int 返回码 0：成功 1：失败
result List 围栏类型集合
