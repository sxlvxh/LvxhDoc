获取区域时间段下人员
请求路径：/cgtasktemptree/getPostTreeUserList.action
接口使用说明： 所有任务地图右侧岗点树查询岗点下人员列表使用
请求参数描述
请求JSON： 
{
    "entCode": "320101",   //单位编码，必填
    "deptId": 320101,      //部门ID，非必填
    "areaId":null,         //任务区域ID，非必填
    "areaType":"1",        //电子围栏类型，必填
    "efenceId":"1",        //电子围栏ID，必填
    "startTime":"",        //任务开始时间，非必填
    "endTime":"",          //任务结束时间，非必填
    "cgTemplateType":"1",  //任务类型，1日常任务，2保障任务，3临时任务4上报事件5渣土管控，必填
    "classify":""          //保障任务类型，cgTemplateType为2时必填，1扫雪防冻2国家公祭日3灯会保障4防汛保障
}
返回参数描述：
属性 类型 备注 说明
code int 返回码 0：成功 1：失败
result List 执行人集合
