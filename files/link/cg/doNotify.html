录像回调通知
请求路径：/videonotify/doNotify.action
请求参数描述
属性 类型 备注 说明
busiType String startRecord 开始录像透传参数 locRecordUpload 本地录像上传通知 screenRecord 开始屏幕录像
strTrunkPara String 需要base64编码 透传参数
请求JSON： 
{
    "msgID": 20112,
    "msgTitile": "default",
    "msgbody": {
        "strDomainCode": "default",
        "nRecord": 1,
        "nSessionID": 1,
        "strTrunkPara": "default"
    }
}
strTrunkPara 的格式进行说明 
{
    "busiType": "startRecord", 
"object": {
        "eventId": "100015", 
        "execLoginName": "zx01", 
        "execId": "100005",
        "callCode": "1111",--备用字段，
    "entCode":"10000" ,
    }
}
返回参数描述：
属性 类型 备注 说明
code int 返回码 0：成功 1：失败
desc String 返回描述
返回JSON： 
{
  "code": 0,
  "desc": "操作成功"
}
