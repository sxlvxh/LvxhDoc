根据岗点或者网格查询任务列表(废弃)新接口
请求路径：/cgtasktemplatelist/getTaskListByPostOrGrid.action
请求参数描述
属性 类型 备注 说明
entCode String 必须输入 单位代码
postId int 必填 岗点ID
gridId int 非必填 网格ID
请求JSON： 
{
	"entCode":"320101",
	"postId":17,
	"gridId": 21  //此参数不传查岗点下所有任务，传此参数则值查询刚网格下所有任务
}
返回参数描述：
属性 类型 备注 说明
code int 返回码 0：成功 1：失败
result List 任务集合
