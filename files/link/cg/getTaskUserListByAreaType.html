根据围栏类型查询人员列表
请求路径：/cgtasktemplatelist/getUserListByAreaType.action
接口使用说明： 场景描述：地图上点击岗点弹出任务列表，在点击时间回放，左边树形查询对应岗点下的人员列表
请求参数描述
属性 类型 备注 说明
entCode String 必须输入 单位代码
areaType String 必填 围栏类型
efenceId int 必填 电子围栏ID
taskDate String 非必填 任务日期
请求JSON： 
{
	"entCode":"320101",
	"areaType":"2",
	"efenceId":17,
	"taskDate":"2018-12-26"
}
返回参数描述：
属性 类型 备注 说明
code int 返回码 0：成功 1：失败
result List 人员集合
